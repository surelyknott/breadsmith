# The Breadsmith

Marketing website starter for a local microbakery, built with React, Vite, and React Router.

## Stack

- React 19
- Vite
- React Router
- Plain CSS

## Run locally

```bash
npm install
npm run dev
```

## Available scripts

- `npm run dev` starts the local development server
- `npm run build` creates a production build
- `npm run lint` runs ESLint
- `npm run preview` serves the production build locally

## Content edits

Most placeholder business content lives in `src/siteData.js`.
